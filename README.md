# 医疗器械试剂耗材快速下单系统

一个现代化、用户体验极佳的医疗器械试剂耗材快速下单Web界面。

## 功能特点

### 🎯 核心功能
- **快速搜索** - 支持产品名称、型号、规格的实时搜索
- **分类筛选** - 按产品类别快速筛选，包括防护用品、检测试剂、注射器械等
- **智能购物车** - 实时更新，支持数量调整，本地存储保存
- **快速下单** - 简洁明了的订单提交流程
- **订单导出** - 支持将订单导出为JSON格式

### 💎 用户体验优化
- **响应式设计** - 完美适配桌面端和移动端
- **现代化UI** - 美观的渐变色彩和流畅的动画效果
- **直观操作** - 清晰的产品卡片、购物车侧边栏、订单确认弹窗
- **实时反馈** - 操作提示、加载状态、成功消息等
- **数据持久化** - 购物车数据自动保存到本地存储

### 🏥 产品分类
- 防护用品（口罩、手套、隔离衣、护目镜等）
- 检测试剂（PCR试剂盒、血糖试纸、血常规试剂等）
- 注射器械（注射器、输液器等）
- 消毒用品（酒精、碘伏等）
- 采血器械（采血管、病毒采样管等）
- 手术器械（手术刀片等）
- 实验耗材（培养皿等）
- 敷料用品（胶带、棉签等）
- 监测器械（心电电极片等）

## 使用方法

### 快速开始

1. **直接打开**
   - 直接用浏览器打开 `index.html` 文件即可使用

2. **本地服务器（推荐）**
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 或使用Node.js
   npx serve .
   
   # 然后访问 http://localhost:8000
   ```

### 操作指南

#### 浏览产品
- 在顶部搜索框输入关键词快速查找产品
- 点击左侧分类菜单筛选不同类别的产品
- 使用排序下拉框按名称或价格排序

#### 添加到购物车
- 点击产品卡片上的"加入购物车"按钮
- 产品会自动添加到购物车，右上角购物车图标会显示商品数量

#### 管理购物车
- 点击右上角购物车图标打开购物车侧边栏
- 在购物车中可以：
  - 调整商品数量（使用+/-按钮或直接输入）
  - 删除商品（点击垃圾桶图标）
  - 查看总价
  - 清空整个购物车

#### 下单流程
1. 确认购物车中的商品和数量
2. 点击"快速下单"按钮
3. 填写订单信息：
   - 联系人姓名（必填）
   - 联系电话（必填）
   - 单位/机构（必填）
   - 收货地址（必填）
   - 备注信息（选填）
4. 确认订单明细和总价
5. 点击"提交订单"完成下单

#### 导出订单
- 点击侧边栏的"导出订单"按钮可将当前购物车导出为JSON文件

## 自定义和扩展

### 添加新产品
在 `script.js` 文件的 `productsData` 数组中添加新对象：

```javascript
{
    id: 19,
    name: '产品名称',
    category: '分类名称',
    spec: '规格说明',
    price: 价格,
    stock: 库存数量,
    image: '图片类型标识'
}
```

### 修改样式
- 在 `styles.css` 中修改 CSS 变量来调整主题色彩
- 所有主要颜色都在 `:root` 中定义
